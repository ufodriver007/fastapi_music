## Демо приложение музыкальный сервис
#### Описание
>SPA-приложение музыкальный сервис 
###### Технологии
- Бекенд: Python + FastAPI + PostgreSQL + Redis + NGINX
- Фронт: JS + React + HTML + CSS + Tailwind
- Docker
###### Использование
- Создание пользователя
- Поиск песен (используются сервера mail.ru, но можно давить и любой другой подобный сервис)
- Создание/удаление/переименование плейлистов
- Прослушивание/переменование/удаление песни
- Сортировка песен в плейлисте
#### Запуск
1. Создаём файл `.env` с переменными окружения по аналогии с файлом `.example.env`
2. 
```angular2html
docker compose up
```
